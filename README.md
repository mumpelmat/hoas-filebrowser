# Local File Browser for Home Assistant OS

Private Home Assistant OS add-on for basic file operations in mounted HAOS folders.

## Features

- Browse allowed folders: config, share, media, ssl, addons, backup
- Upload files from desktop/browser
- Download files
- Create folders
- Rename files/folders
- Copy files/folders
- Move files/folders
- Delete files/folders
- Integrated via Home Assistant Ingress

## Installation as local add-on

1. Copy the folder `local_file_browser` into Home Assistant:
   `/addons/local_file_browser`
2. In Home Assistant go to:
   Settings → Add-ons → Add-on Store → ⋮ → Repositories / reload local add-ons
3. Install `Local File Browser`
4. Start the add-on
5. Open it from the sidebar or add-on page

## Warning

This add-on can modify important Home Assistant files. Use carefully and keep backups.
